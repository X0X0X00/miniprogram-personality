# 恋爱人格测试 · 微信小程序

18道单选题，测出你的恋爱人格类型。纯本地计算，无后端。

![预览](avatar.png)

## 功能

- 18题问卷，覆盖6个恋爱维度
- 客户端计分，匹配16种人格类型
- 结果页一键生成分享图，保存到相册

## 本地开发

1. 克隆仓库

```bash
git clone https://github.com/X0X0X00/miniprogram-personality.git
```

2. 打开**微信开发者工具**，导入 `miniprogram-personality/` 目录，填入你的 AppID

3. 点击编译即可预览

## 人格维度

| 维度 | 低端（L） | 高端（H） |
|------|---------|---------|
| 亲密主动性 | 等待型 | 主动型 |
| 情感深度 | 理性型 | 感性型 |
| 边界感 | 融合型 | 独立型 |
| 安全感 | 稳定型 | 自由型 |
| 表达方式 | 内敛型 | 外放型 |
| 关系目标 | 陪伴型 | 成长型 |

每个维度根据该维度3道题的得分之和判断 H/L（≥2 为 H），6位组合精确匹配16种人格类型，无精确匹配时取汉明距离最近的类型。
